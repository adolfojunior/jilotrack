Demonstração jCompany
Esta aplicação reúne uma série de conceitos e padrões de implementação Java EE 6 inovadores. Para seu melhor entendimento, sugerimos que seja seguido o roteiro abaixo:
1
Consulte a Ajuda On-Line do Eclipse ou clique aqui para uma introdução geral sobre o jCompany Developer Suite e frameworks Open-Source de base utilizados para construir esta aplicação
2
Clique nos links a seguir para consultar documentação sobre algumas APIs chave do Java EE 6, utilizadas nesta aplicação: CDI 1.0 (JSR 299, JBoss Weld 1.0); Bean Validation 1.0 (JSR 303, Hibernate Validator 4.0); JPA 2.0 (JSR 317, Hibernate Core 3.5.1); JSF 2.0 (JSR 314, Apache Trinidad Alpha2); JAX-RS 1.0 (JSR 311, JBoss Weld 1.0); JAX-WS 2.0 (JSR 224);
3
Abra os códigos fontes e estude-os. Esta aplicação utiliza basicamente código Java para Entidades de Domínio e regras de negócio. Todos os demais artefatos são generalizados via programação OO ou gerados (artefatos "não Java" como XHTML e rótulos)
4
Se, ainda assim, persistirem dúvidas, contate a Powerlogic utilizando o email suporte@powerlogic.com.br. Descreva sua dúvida, informando versão e dados para contato.
